FEMhub Online Lab
=================

This repository (femhub-online-lab) contains open source
code for the FEMhub Online Laboratory. The Online Lab is an
advanced web application written in ExtJS whose goal is to 
make scientific computing accessible to anyone, without 
the need for expensive hardware or software. The Online Lab 
operates inside the web browser window, that turns into 
a virtual computer desktop. The Online Lab is accessible 
from PCs, laptops, netbooks, iPads, PDAs, and even web-enabled 
smart phones.
